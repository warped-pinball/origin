Layout tokens

Use the shared background and spacing variables to keep admin tools consistent with marketing pages.

  • Backgrounds

    --page-background (spiral sky) · --page-background-plain (white)

    Apply the plain surface to focused tools like the machine admin view; keep the spiral base for public-facing pages.

  • Surfaces

    --page-surface · --page-surface-muted

    Cards, hero strips, and inline sections rely on these values for readable contrast.

  • Spacing + shells

    --page-max-width (1200px) · --page-gutter (24px) · --space-050…1200

    Wrap content with page-shell or c-shell and use the space scale to balance sections.

Typography

Pair display headers set in Lilita One with Inter for everything else.

  • Headlines

    Lilita One, 46px, 1.05 line height

    Used on page titles and card titles that lead sections.

  • Body

    Inter, 16–18px, 1.5 line height

    Default font for paragraphs, table text, and labels.

  • Labels

    Inter Bold, uppercase, letter spacing 0.04em

    Apply to nav links, buttons, and eyebrows.

HTML + CSS

<h1 class="display-title">Warped Pinball League</h1> <p class="body-text">Inter keeps long-form content comfortable to read.</p> .display-title { font-family: 'Lilita One', cursive; font-size: 46px; line-height: 1.05; } .body-text { font-family: 'Inter', system-ui, sans-serif; font-size: 18px; line-height: 1.5; }

Cards and layout

Cards keep content contained, with bold borders and playful shadows.

Spacing

24px padding, 22px border radius, 4px border

Use card for primary content areas and card--aside for sticky support panels.

Page shell

Max width 1200px with 24px gutter

Wrap screens with page-shell to keep layouts centered and breathable.

HTML

<main class="page-shell layout"> <article class="card">Primary content</article> <aside class="card card--aside">Helper panel</aside> </main>

Buttons

Buttons are chunky, bright, and provide clear hover feedback.

  • Shape

    14px radius, 3px border, inset shadow

    Keep inline padding at 14–18px and use bold labels.

  • Hover

    Lift by 1px with accent fill

    Maintain high contrast between the label and background.

HTML

<button class="button" type="submit">Save changes</button> <button class="button button--ghost" type="button">Cancel</button>

Forms

Fields are roomy with generous spacing and clear inline validation.

Form rows use a two-column grid on desktop and stack on smaller screens.

HTML

<label class="field field--error"> <span>Email</span> <input type="email" aria-invalid="true" placeholder="player@example.com"> </label>
